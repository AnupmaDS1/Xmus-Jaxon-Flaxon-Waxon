# Xmus-Jaxon-Flaxon-Waxon

This project investigates whether crime type and neighbourhood (income and employment) deprivation indicators can be used to predict the outcome of a recorded crime, and whether deprivation adds predictive value beyond crime type alone. 

Limitations in the data available influence this choice of question. The data lacks important details such as evidence quality, victim cooperation and investigative resources that may explain outcomes. Consequently, the project focuses on prediction rather than causal explanation, recognising that any relationships identified may reflect unobserved underlying factors we don’t have the data for.
